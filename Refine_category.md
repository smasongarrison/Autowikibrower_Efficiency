# Useful Skips for adding folks to a specific century

## 20th century
### Text Contains

```
Category:(?:20(?:[1-9]\d|0[2-9]) deaths|Living people|British (?:psychiatrists|consciousness researchers and theorists))
```
### Doesn't Contain

```
Category:(19\d\d|18[89]\d) births
```
